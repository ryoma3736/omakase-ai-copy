本物のOmakase.aiから徹底的にスクレイピングした正確な機能リスト。

## コア機能

### AI Sales Agent 能力

| 機能 | 説明 |
|------|------|
| **Multi-Modal AI Agent** | 音声、チャット、ビジュアル、サイト閲覧をすべて同時に対応。ページコンテキストを理解しリアルタイムに適応 |
| **Drives Real Sales** | 単なるサポートチャットボットではない。顧客の発見、決定、購入を積極的に支援し、スマートなレコメンデーションでコンバージョン向上 |
| **Hyper-Personalized** | 対話ごとにAIが学習・改善し、カジュアルな訪問者をロイヤルカスタマーに変換 |
| **Built for Any Business** | 個人ショップから大規模小売まで、技術スキル不要でスケール |
| **On-Brand** | AIエージェントのペルソナをブランドのトーンやスタイルにカスタマイズ |
| **Voice-Powered Sales** | 本物の営業担当のように説得力があり、常にオンブランド。スクリプト化されたチャットボットではない |
| **Live Product Recommendations** | 会話中および顧客がサイトを閲覧中にリアルタイムで適切な商品を提案 |

### セットアップ & 統合

| 機能 | 説明 |
|------|------|
| **Effortless Setup** | WebサイトURLを入力するだけ。コード不要で数分で開始 |
| **AI-Powered Scraping** | AIがサイトから自動学習。セットアップ遅延やコンテンツ整理不要 |
| **Shopify Integration** | Shopifyアプリ統合。商品、決済等をリンク |
| **Easy to Train** | Agentic Dashboardでナレッジ管理、AIエージェントの微調整。ファイルアップロード対応 |
| **Customizable Widget** | ウィジェットの動作と外観をサイトデザインやカスタマージャーニーに合わせて調整 |

### アナリティクス

| 機能 | 説明 |
|------|------|
| **Conversational Analytics** | 実際の会話データからAIパワードインサイト。顧客の要望や購買行動を理解 |
| **Dashboard** | 顧客が何を質問し、何を探し、何を購入しているかを一覧表示 |
